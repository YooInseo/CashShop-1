# 💸 캐시 + 캐시상점 API 💸


## ⚙️ 개발 환경 ⚙️
- JDK(Java Development Kit) 17
- SPIGOT 1.19.2


## 💵 캐시 활용하기 💵
```java
public class Example() {
  CashAPI cashAPI = new CashAPI();

  cashAPI.depositCash(target, amount);
  cashAPI.withdrawCash(target, amount);
  cashAPI.setCash(target, amount);
  cashAPI.getCash(target)
}

```
